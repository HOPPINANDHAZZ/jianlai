关于为什么重构

1、本博客项目前后端分离，后端采用微服务架构，将原有项目的登录服务、博客服务、文件服务、搜索服务、爬虫服务拆分，并将文件服务单独部署在一台云服务器上。

2、集成剑来框架的网关，并将服务注册于剑来的注册中心，来实现服务的管理和远程调用(rpc)

亮点

完美的页面

每一个组件，每一个布局都是精心打磨，兼容各种分辨率。并为编写博客集成3大编辑器

数据库读写分离

虽然很主流，但好像并无卵用，采用mycat作为数据同步中间件

强大的搜索引擎

使用Lucene作为检索引擎。Lucene是ES、hadoop等的底层框架，自带4种分词器，可以给任意字段添加索引且搜索速度极快

redis为你保驾护航

不用害怕因为意外导致白写，redis每时每刻都会记录页面各种状态
