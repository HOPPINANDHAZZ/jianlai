    栅格配置

    Bootstrap使用ems或rems定义长度大小， 使用px用于网格和容器宽度。 因为宽度以像素为单位，不随字体大小而改变

    Extra small
    <576px     		Small
    ≥576px     		Medium
    ≥768px     		Large
    ≥992px     		Extra large
    ≥1200px
    容器最大宽度 无 (自动) 540px 720px 960px 1140px
    class前缀 .col- .col-sm- .col-md- .col-lg- .col-xl-
    最大列数 12
    间距(排水沟)宽度 30px (两侧各15px)
    是否允许可嵌套 是
    列排序 是

    响应列重置

    由于有四层网格可用，您肯定会遇到一些问题，在某些地方，由于一个列比另一个列高， 您的列无法完全清除。要解决这个问题，请结合使用类标签.clearfix清除浮动来调整的好看。
    .col-xs-6 .col-sm-3
    调整窗口大小或在手机上查看这个案例
    .col-xs-6 .col-sm-3
    .col-xs-6 .col-sm-3
    .col-xs-6 .col-sm-3

								<div class="col-xs-6 col-sm-3"> .col-xs-6 .col-sm-3

    偏移列

    为列添加偏移类 .offset-md-*。 通过为 * 设置一个数字增加margin. 例如, .offset-md-4 class会使 .col-md-4 偏移4列。
    .col-md-4
    .col-md-4 .col-md-offset-4
    .col-md-3 .col-md-offset-3
    .col-md-3 .col-md-offset-3
    .col-md-6 .col-md-offset-3
    						
							<div class="row bg-light m-b-10 p-10">
								<div class="col-md-4">.col-md-4</div>
								<div class="col-md-4 offset-md-4">.col-md-4 .col-md-offset-4</div>
							</div>
							<div class="row bg-light m-b-10 p-10">
								<div class="col-md-3 offset-md-3">.col-md-3 .col-md-offset-3</div>
								<div class="col-md-3 offset-md-3">.col-md-3 .col-md-offset-3</div>
							</div>
							<div class="row bg-light m-b-10 p-10">
								<div class="col-md-6 offset-md-3">.col-md-6 .col-md-offset-3</div>
							</div>
