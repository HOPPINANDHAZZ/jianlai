    颜色盒子

    2. UI
    3. 盒子
    4. 颜色盒子

    背景颜色

    每个盒子都可以额外添加class .box-* 来使用颜色，其中 * 是 颜色名称 如要用绿色的盒子，可以使用class .box-success 放在 .box 标签上。由于添加了背景颜色，需要额外使用class.box-inverse让字体颜色设为白色

    Primary

    在盒子.box上，添加class.box-primary，如需让字体变为白色，需要额外使用class.box-inverse

    Secondary 没有使用 .box-inverse

    在盒子.box上，添加class.box-secondary，如需让字体变为白色，需要额外使用class.box-inverse，这里就没用

    Success

    在盒子.box上，添加class.box-success，如需让字体变为白色，需要额外使用class.box-inverse

    Info

    在盒子.box上，添加class.box-info，如需让字体变为白色，需要额外使用class.box-inverse

    Warning

    在盒子.box上，添加class.box-warning，如需让字体变为白色，需要额外使用class.box-inverse

    Danger

    在盒子.box上，添加class.box-danger，如需让字体变为白色，需要额外使用class.box-inverse

    实心盒子头部

    每个盒子都可以添加class .box-solid .bg-* 来实现下面效果，其中 * 是颜色名称

    Primary

    在盒子.box上，添加class.box-solid .box-primary即可

    Secondary

    在盒子.box上，添加class.box-solid .bg-secondary即可

    Success

    在盒子.box上，添加class.box-solid .bg-success即可

    Info

    在盒子.box上，添加class.box-solid .bg-info即可

    Warning

    在盒子.box上，添加class.box-solid .bg-warning即可

    Danger

    在盒子.box上，添加class.box-solid .bg-danger即可

    盒子轮廓

    每个盒子都可以添加class .box-outline-*来添加颜色轮廓 * 是颜色名称

    Primary

    在盒子.box上，添加class.box-outline-primary即可

    Secondary Without .box-inverse

    在盒子.box上，添加class.box-outline-secondary即可

    Success

    在盒子.box上，添加class.box-outline-success即可

    Info

    在盒子.box上，添加class.box-outline-info即可

    Warning

    在盒子.box上，添加class.box-outline-warning即可

    Danger

    在盒子.box上，添加class.box-outline-danger即可

    单边框盒子

    你可以在盒子的任何一侧添加边框。在某些情况下，你可能只希望在盒子的一侧有一个边框。

    左 边界

    .bl-3.border-primary

    需要结合bl-*来实现，就是border-left的简写，然后额外使用class.border-primary来改变颜色

    上 边界

    .bt-3.border-info

    需要结合bt-*来实现，就是border-top的简写，然后额外使用class.border-info来改变颜色

    下 边界

    .bb-3.border-warning

    需要结合bb-*来实现，就是border-bottom的简写，然后额外使用class.border-warning来改变颜色

    右 边界

    .br-3.border-danger

    需要结合br-*来实现，就是border-right的简写，然后额外使用class.border-danger来改变颜色
