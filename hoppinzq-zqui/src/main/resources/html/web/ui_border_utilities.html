    可以使用以下class添加边框。您可以使用border color的class更改边框颜色。

    class的规则 b{side}-{size} 格式, 其中 {size}是下面的几个： 0, 1, 2, 3, 4, 或 5 。其中 {side} 是一下几个：

    • t: Border top
    • b: Border bottom
    • l: Border left
    • r: Border right
    • x: Border left and right
    • y: Border top and bottom
    • null: 全部方位的边框
