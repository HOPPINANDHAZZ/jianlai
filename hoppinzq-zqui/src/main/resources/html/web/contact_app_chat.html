    一种是负载均衡算法，通过设置权重来实现。nginx就是基于该算法实现的负载均衡。 每次请求都会通过负载均衡算法根据权值转发请求。 假设有1个服务分别部署在三台服务器上，假设该服务有一个接口queryUesr用于查询用户信息。 通过配置nginx.conf给每个服务设置随机权值1，2，3，总权值是6。 当请求进入nginx后先判断哪个服务权值高，相同权值的服务将会随机选择一个服务转发请求。 然后所有的服务在它的权值基础上加其本身的权值，此时的权值是2，4，6。由于总权值是6， 此时权值大于等于6的权值会减去总权值，所以第一次调用后三个服务的queryUser接口的权值是2，4，0。 第二次请求就来就是3，6，3，转发至第二个服务后变成3，0，3。然后是4，2，6，再调用第三个服务4，2，0。
