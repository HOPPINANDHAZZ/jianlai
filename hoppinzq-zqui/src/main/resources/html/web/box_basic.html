    基本盒子，没有头部标题

    呃，最基本的盒子，为div添加class.box即可使用盒子的一系列样式。 以下案例必须要求父div有.box，盒子内容需要使用.box-body

    盒子标题

    h4.box-title

    选择器是 h4.box-title 希望你能看懂，就是class为box-title的h4标签

    盒子标题 大小

    h1.box-title h2.box-title h3.box-title h4.box-title h5.box-title

    使用h标签添加classbox-title就行

    盒子头部class box-header

    .box-header > .box-title

    盒子头部就是使用class.box-header即可，然后标题可以放在其中。会有一道杠杠来分割头部跟身体

    盒子头部 不使用边框(底部)

    .box-header .without-border

    不想要那道杠杠？在头部.box-header添加class.without-border即可

    标头加粗

    .box-title-bold

    在标题.box-title添加class.box-title-bold即可

    标题加 描述 我是小小的描述

    .box-title <small>我是小小的描述</small>

    在标题.box-title后追加<small>我是小小的描述</small>即可

    标题和 副标题1 我是副标题1

    .sidetitle

    在标题描述small中添加class.sidetitle即可

    标题和 副标题2 我是副标题2

    .subtitle

    在标题描述small中添加class.subtitle即可

    边框 盒子

    .box-bordered

    在盒子.box中添加class.box-bordered即可添加边界,可选择额外添加class.border-*来改变颜色。

    阴影 盒子

    .box-shadowed

    在盒子.box中添加class.box-shadowed即可添加边界上的阴影

    盒子移入阴影并突出

    .box-hover-shadow .pull-up

    在盒子.box中添加class.box-hover-shadow即可添加移入的阴影 ，额外添加class.pull-up即可添加移入的突出。 可再添加额外添加class.no-shadow去掉阴影。

    透明的 盒子

    .box-transparent

    在盒子.box中添加class.box-transparent即可实现透明,最好再额外添加class.no-shadow去掉阴影。

    盒子底部 box-footer

    div.box-footer

    在.box的div内，在一个div上使用class .box-footer即可

    我是页脚

    盒子底部 案例4
